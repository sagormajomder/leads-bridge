# LeadsBridge (FINTRIXX)

LeadsBridge is a modern, responsive landing page for an Amazon Growth Partner agency. The agency helps Amazon sellers increase sales, optimize ads, rank higher, manage inventory, and expand into new markets.

## Features

- **Responsive Design**: Fully responsive across mobile, tablet, and desktop devices.
- **Engaging UI/UX**: Includes a dynamic hero section with floating statistics cards and gradient backgrounds.
- **Services Overview**: Clearly outlines core services such as PPC Optimization, SEO & Product Ranking, Inventory Management, and Scaling.
- **Calendly Integration**: Allows potential clients to book a free strategy call directly from the landing page.
- **Custom Styling**: Built with modern CSS practices for clean and modular design.

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript Vanilla**
- **Google Fonts**

## Run it Locally

To view and interact with the project locally:

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.

_No local development server (like Node.js) or build tools are required._
